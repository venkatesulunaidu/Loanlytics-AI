================================================================================
  SETUP (5 MINUTES)
================================================================================

STEP 1: Install Dependencies
-----------------------------
pip install -r requirements.txt


STEP 2: Create Configuration
-----------------------------
Copy config.example to .env:

    Copy-Item config.example .env

Then edit .env with your credentials:

    MYSQL_HOST=localhost
    MYSQL_USER=your_mysql_username
    MYSQL_PASSWORD=your_mysql_password
    MYSQL_DATABASE=your_database_name
    OPENAI_API_KEY=sk-your-openai-api-key
    OPENAI_MODEL=gpt-4-turbo-preview

Get OpenAI Key: https://platform.openai.com/api-keys


STEP 3: Start the Application
-----------------------------

OPTION A - Automated (Easy):
    Double-click: START.bat

OPTION B - Manual:
    Terminal 1 (Backend):
        python backend/app.py

    Terminal 2 (Frontend):
        Open frontend/index.html in your browser


================================================================================
  HOW TO USE
================================================================================

1. Open frontend/index.html in your web browser

2. Wait for "Connected" status (green dot)

3. Type your question in plain English:
   - "Show me top 10 customers with highest loan amounts"
   - "How many loans were disbursed in 2023?"
   - "What is the average interest rate?"

4. Click "Ask Question"

5. View the generated SQL and results

6. Export to CSV if needed

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: Backend won't start
Solution: 
    - Check .env file exists
    - Verify MySQL credentials
    - Ensure MySQL is running

Problem: Frontend shows "Backend Offline"
Solution:
    - Start backend: python backend/app.py
    - Check backend is on http://localhost:5000

Problem: "Token limit exceeded"
Solution:
    - In .env, use: OPENAI_MODEL=gpt-4-turbo-preview
    - This model handles large databases (128k tokens)

Problem: "Can't connect to database"
Solution:
    - Check MYSQL_* values in .env
    - Verify database exists
    - Test MySQL connection
